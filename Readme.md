# 📱 Cellphone Price Range Prediction

## Project Overview
This project builds a Machine Learning model to predict the price range of mobile phones based on specifications such as RAM, battery power, camera quality, and screen resolution.

Price Categories:
0 → Low Cost 
1 → Medium Cost 
2 → High Cost 
3 → Very High Cost 

## Problem Statement
A mobile company wants to estimate the price range of phones based on their features. Using historical mobile phone data, we build a model that predicts the correct price category.

## Dataset Features
- battery_power – Battery capacity (mAh)
- blue – Bluetooth support
- clock_speed – Processor speed
- dual_sim – Dual SIM support
- fc – Front camera megapixels
- four_g – 4G support
- int_memory – Internal memory (GB)
- mobile_wt – Mobile phone weight
- n_cores – Number of processor cores
- pc – Primary camera megapixels
- px_height – Pixel resolution height
- px_width – Pixel resolution width
- ram – RAM in MB
- sc_h – Screen height
- sc_w – Screen width
- talk_time – Maximum talk time
- three_g – 3G support
- touch_screen – Touch screen support
- wifi – WiFi support

Target Variable:
price_range
